## 第1讲 命题逻辑基础 答案

### 1.1 命题判断
**命题**：1, 3  
**非命题**：2（疑问句）、4（含自由变量 x）、5（感叹句）、6（悖论）
> 🔵 "我正在说假话" 是悖论，**不是命题**（无确定真值）

### 1.2 符号化
1. P → Q
2. Q → P
3. P ∧ ¬Q
4. P ∨ ¬Q（"除非 P 否则 ¬Q" ⇔ "P ∨ ¬Q"）
5. Q → P

### 1.3 公式类型
1. (P → Q) ↔ (¬P ∨ Q)：**永真式**（化归公式）
2. (P ∧ Q) → (P ∨ Q)：**永真式**
3. ¬(P → Q) ∧ Q：可满足式（P=F, Q=T 时为 T）

### 1.4 真值表
1. (P ∧ ¬Q) ∨ (¬P ∧ Q) 是 P、Q 的异或
2. P → (Q → R)：仅当 P=T, Q=T, R=F 时为 F

### 1.5 排中律 / 矛盾律
- P ∨ ¬P：所有赋值下为 T → 永真
- P ∧ ¬P：所有赋值下为 F → 永假

### 1.6 HS 真值表证明
8 行真值表，所有赋值下 (P → Q) ∧ (Q → R) → (P → R) 均为 T → 永真

### 1.7 符号化
设 S(x)：x 是班里学生；L(x)：x 学过微积分；F(x)：x 数学不及格
- "每个学生学过微积分"：∀x(S(x) → L(x))
- "至少有一个人数学不及格"：∃x(S(x) ∧ F(x))
- 整体：∀x(S(x) → L(x)) ∧ ∃x(S(x) ∧ F(x))

### 1.8 异或
P⊕Q = (P ∨ Q) ∧ ¬(P ∧ Q)  
为 T 的赋值：(0,1), (1,0)

### 1.9 主合取范式
A(P,Q) = (P ∨ ¬Q) （只在 P=1, Q=0 时为 F，大项 M₁₀）

### 1.10 异或等价
真值表验证：¬(P ↔ Q) 与 (P∧¬Q)∨(¬P∧Q) 真值表完全相同

## 第2讲 等值式 答案

### 2.3 化简
1. ¬(¬P ∨ Q) ∨ (P ∧ ¬Q) = (P ∧ ¬Q) ∨ (P ∧ ¬Q) = P ∧ ¬Q
2. (P ∨ Q) ∧ (P ∨ ¬Q) = P ∨ (Q ∧ ¬Q) = P ∨ 0 = P
3. P ∨ (¬P ∧ Q) = (P ∨ ¬P) ∧ (P ∨ Q) = 1 ∧ (P ∨ Q) = P ∨ Q
   > 🔵 第 3 题用了分配律 + 否定律

### 2.4 证明
(P → Q) ∧ (P → R) = (¬P ∨ Q) ∧ (¬P ∨ R) = ¬P ∨ (Q ∧ R) = P → (Q ∧ R)

### 2.5 证明
P → (Q ∨ R) = ¬P ∨ Q ∨ R  
(P → Q) ∨ (P → R) = (¬P ∨ Q) ∨ (¬P ∨ R) = ¬P ∨ Q ∨ R  
两者相同 ✓

### 2.9 范式
(P → Q) → R = ¬(¬P ∨ Q) ∨ R = (P ∧ ¬Q) ∨ R
- **析取范式**：(P ∧ ¬Q) ∨ R
- **合取范式**：(P ∨ R) ∧ (¬Q ∨ R)

### 2.10 证明
P ↔ Q = (P → Q) ∧ (Q → P) = (¬P ∨ Q) ∧ (¬Q ∨ P)  
展开：(¬P ∧ ¬Q) ∨ (¬P ∧ P) ∨ (Q ∧ ¬Q) ∨ (Q ∧ P) = (¬P ∧ ¬Q) ∨ (P ∧ Q) ✓

## 第3讲 重言蕴涵与推理 答案

### 3.2 MT 拒取式
```
(1) P → Q    P  前提
(2) ¬Q       P  前提
(3) ¬P       T(1)(2) MT
```

### 3.3 HS 假言三段论
```
(1) P → Q    P
(2) Q → R    P
(3) P        P
(4) Q        T(1)(3) I 假言推理
(5) R        T(2)(4) I
```

### 3.6 推理判断
| 推理 | 判断 | 说明 |
|---|---|---|
| P→Q, Q ∴ P | ❌ | 肯定后件谬误 |
| P→Q, ¬P ∴ ¬Q | ❌ | 否定前件谬误 |
| P∨Q, ¬P ∴ Q | ✅ | 析取三段论 |
| P∧Q ∴ P↔Q | ✅ | 合取 → 等价 |

> 🔴 推理题最容易错的是"肯定后件"和"否定前件"，一定要记牢。

### 3.7 玩扑克题
设 P：我学习；Q：我数学及格；R：我热衷于玩扑克
前提：P→Q, ¬R→P, ¬Q，求证：R

```
(1) P → Q       P
(2) ¬R → P      P
(3) ¬Q          P
(4) ¬P          T(1)(3) MT
(5) ¬¬R         T(2)(4) MT    (¬R→P 的逆否：¬P→R)
(6) R           T(5) 化简
```
> 🔵 "¬R→P" 的逆否是 "¬P→R"，再用 MT 得到 R

### 3.8 真话者
设甲说真话 P。则甲是"我们俩都是假话者"为真 → 乙是假话者。  
但 P 是甲说真话，与乙是假话者不矛盾。再设 P 乙说"至少有一个真话者"为真，至少有一个真话者即甲真，✓  
但继续推："至少一个"包括甲乙都真情况。如果甲乙都真，则甲说"我们俩都是假话者"应为假（因为我们俩都真），矛盾。

所以 **甲乙都是假话者**。
> 🔴 这种题用真值表枚举 4 种情况最快。

### 3.9 凶手（课件原题）
逐步推理：
1. G = T（经理有钱）
2. H = F（A 不富裕）
3. ¬E（灯灭）→ D = T（"B 证词不正确"会推出"灯未灭"，但灯灭了，故 B 证词正确）
4. D = T → C = T（午夜前谋害）
5. C = T → ¬A（"A 谋害" 蕴含 "不在午夜前"，故 A 未谋害）
6. A ∨ B = T → B = T

**结论：秘书 B 是凶手** 🔪

### 3.10 推理不成立
原推理：P→Q, ¬R∨P, Q ⇒ R→S 不成立
**原因**：前提与 R→S 没有关联  
**修正**：需要前提 S 才能推出 R→S

## 第4讲 范式 答案

### 4.1 范式
(P → Q) ∧ R = (¬P ∨ Q) ∧ R
- **析取范式**：(¬P ∧ R) ∨ (Q ∧ R)
- **合取范式**：(¬P ∨ Q) ∧ R

### 4.2 (P↔Q) → R
(P↔Q) → R = ¬(P↔Q) ∨ R = ((P∧¬Q) ∨ (¬P∧Q)) ∨ R
- **主析取范式**：m₀₁₀ ∨ m₁₀₀ ∨ m₀₀₁ ∨ m₀₁₁ ∨ m₁₀₁ ∨ m₁₁₁  
  = (¬P∧Q∧¬R) ∨ (P∧¬Q∧¬R) ∨ (¬P∧¬Q∧R) ∨ (¬P∧Q∧R) ∨ (P∧¬Q∧R) ∨ (P∧Q∧R)
- **主合取范式**：M₀₀₀ ∧ M₁₁₀  
  = (P∨Q∨R) ∧ (¬P∨¬Q∨R)

### 4.3 合取范式
1. (P ∨ ¬Q) → R = ¬(P ∨ ¬Q) ∨ R = (¬P ∧ Q) ∨ R = (¬P ∨ R) ∧ (Q ∨ R)
2. P ∧ (Q ↔ R) = P ∧ ((¬Q ∨ R) ∧ (Q ∨ ¬R)) = (P ∧ ¬Q ∧ ¬Q ∨ R) ... 

### 4.4 P → (Q ∧ R) 主析取范式
真值表：仅 (0,0,0),(0,1,1),(1,1,1) 为 T  
= m₀₀₀ ∨ m₀₁₁ ∨ m₁₁₁ = (¬P∧¬Q∧¬R) ∨ (¬P∧Q∧R) ∨ (P∧Q∧R)

### 4.5 P⊕Q 主合取范式
P⊕Q 真值：(0,0)=F, (0,1)=T, (1,0)=T, (1,1)=F  
为 F 的赋值：(0,0),(1,1)  
= M₀₀ ∧ M₁₁ = (P∨Q) ∧ (¬P∨¬Q)

### 4.6 A(P,Q,R) 双范式
看题给真值表为 T 的赋值：(0,0,0),(0,1,1),(1,0,0),(1,1,0),(1,1,1)
- **主析取**：m₀₀₀ ∨ m₀₁₁ ∨ m₁₀₀ ∨ m₁₁₀ ∨ m₁₁₁
- **主合取**：M₀₀₁ ∧ M₀₁₀ ∧ M₁₀₁  
  (为 F 的赋值：(0,0,1),(0,1,0),(1,0,1))
> 🔵 找小项 → T 行的赋值；找大项 → F 行的赋值

### 4.7 (P ∧ Q) ⇔ (¬P ∨ ¬Q)
(P ∧ Q) ↔ (¬P ∨ ¬Q) = (P ∧ Q ∧ ¬P ∨ ¬Q) ∨ ((¬P ∧ ¬Q) ∧ (P ∨ Q))  
化简 = (P ∧ Q ∧ (¬P ∨ ¬Q)) ∨ (¬P ∧ ¬Q ∧ (P ∨ Q))  
= (P ∧ Q ∧ ¬P) ∨ (P ∧ Q ∧ ¬Q) ∨ (¬P ∧ ¬Q ∧ P) ∨ (¬P ∧ ¬Q ∧ Q)  
= 0  
**永假式**
> 🔵 直觉：(P ∧ Q) 与 (¬P ∨ ¬Q) 互为否定

### 4.8 P ↔ Q 主合取范式（公式法）
P ↔ Q = (P ∧ Q) ∨ (¬P ∧ ¬Q) → 化为合取范式  
= (P ∨ ¬P) ∧ (P ∨ ¬Q) ∧ (Q ∨ ¬P) ∧ (Q ∨ ¬Q)  
= (P ∨ ¬Q) ∧ (¬P ∨ Q) = M₀₁ ∧ M₁₀

### 4.9 (P→Q) ∧ (Q→R) 主析取范式
(¬P ∨ Q) ∧ (¬Q ∨ R)  
真值表为 T：(0,0,0),(0,1,1),(1,0,0),(1,0,1),(1,1,1)  
= m₀₀₀ ∨ m₀₁₁ ∨ m₁₀₀ ∨ m₁₀₁ ∨ m₁₁₁

### 4.10 永真式主析取范式
P ∨ ¬P 真值表恒为 T，所以主析取范式 = **所有 2ⁿ 个小项的析取**（n=变量数）

## 第5讲 谓词逻辑 答案

### 5.1 符号化
1. ∀x S(x) → L(x)
2. ∀x(S(x) → (C(x) ∨ M(x))) （C=加拿大, M=墨西哥）
3. ∃x ∀y∀z (F(x,y) ∧ F(x,z) ∧ ¬E(y,z)) （"存在 x，他的朋友都互不认识"）

### 5.2 每个学生都有一个朋友
∀x(P(x) → ∃y(F(x,y) ∧ P(y)))

### 5.3 量词顺序
1. ∀x∃y L(x,y)：每个 x 都有 y 与之有 L 关系
2. ∃y∀x L(x,y)：存在 y，所有 x 都与 y 有 L 关系
3. **不成立** ∀x∃y ⇒ ∃y∀x。  
   反例：L(x,y): x < y，∀x∃y x<y 真，但 ∃y∀x x<y 假

### 5.4 量词等值化简
1. ¬∀x(P(x) → Q(x)) = ¬∀x(¬P(x) ∨ Q(x)) = ∃x(P(x) ∧ ¬Q(x))
2. ∀x(P(x) → ∃y Q(x,y)) 无简化（P 中含 x，Q 中含 x,y）

### 5.5 D={a,b,c}，∀x∃y P(x,y)
= ∀a(∃y P(a,y)) ∧ ∀b(∃y P(b,y)) ∧ ∀c(∃y P(c,y))  
= [P(a,a) ∨ P(a,b) ∨ P(a,c)] ∧ [P(b,a) ∨ P(b,b) ∨ P(b,c)] ∧ [P(c,a) ∨ P(c,b) ∨ P(c,c)]

### 5.6 苏格拉底
```
(1) ∀x(P(x) → D(x))    P
(2) P(S)                 P
(3) P(S) → D(S)         T(1) UI
(4) D(S)                 T(2)(3) I
```
> 🔵 UI 规则：∀xA(x) ⇒ A(c)，把 ∀x 脱掉，x 换为个体常项

### 5.7 每个人恰好有一个最好朋友
∀x∃y(F(x,y) ∧ ∀z(F(x,z) → (y=z)))

### 5.8 步行 / 汽车 / 自行车
```
(1) ∀x(W(x) → ¬B(x))    P
(2) ∀x(B(x) ∨ K(x))     P
(3) ∃x ¬K(x)             P
(4) ¬K(c)                 T(3) EI
(5) B(c) ∨ K(c)          T(2) UI
(6) B(c)                  T(4)(5) I   析取三段论
(7) W(c) → ¬B(c)         T(1) UI
(8) ¬W(c)                 T(6)(7) MT
(9) ∃x ¬W(x)              T(8) EG
```

### 5.9 前束范式
1. ∀x P(x) → ∃x Q(x) = ¬∀x P(x) ∨ ∃x Q(x) = ∃x ¬P(x) ∨ ∃x Q(x)  
   前束范式 ≈ ∃y(¬P(y) ∨ Q(z)) （变量标准化后）  
2. ¬(∀x P(x) → ∃y Q(y)) = ∀x P(x) ∧ ¬∃y Q(y) = ∀x P(x) ∧ ∀y ¬Q(y)  
   前束范式：∀x∀y (P(x) ∧ ¬Q(y))

### 5.10 错误推理分析
错误：∀x(P(x) → Q(x)), ∀x Q(x) ⇒ ∃x P(x) 不成立  
反例：D = {a}，P(a)=F, Q(a)=F  
- ∀x(P(x) → Q(x)) = P(a) → Q(a) = F → F = T
- ∀x Q(x) = F
- ∃x P(x) = F  
  前提都真，结论假 → 推理错

**正确推理**：∀x(P(x) → Q(x)), ∃x P(x) ⇒ ∃x Q(x)（正确）
> 🔴 易错点：∀Q(x) 不能推出 ∃P(x)

## 第6讲 集合 答案

### 6.1 元素 / 子集
1. 1 ∈ A ✓
2. {1,2} ⊆ A ✓
3. 4 ∈ A ✗（A 中 4 不是直接元素）
4. {4,5} ⊆ A ✓（{4,5} 是 A 的元素）

### 6.2 P({1,2,3})
= {∅, {1}, {2}, {3}, {1,2}, {1,3}, {2,3}, {1,2,3}}，共 8 个 = 2³

### 6.3 化简
A ∪ (A ∩ B) ∪ (A ∩ B ∩ C) = A（吸收律）

### 6.4 集合运算
1. A ∪ B = {1, 2, 3, 4}
2. A ∩ B = {3}
3. A − B = {1}
4. A ⊕ B = {1, 2, 4} （A∪B − A∩B）
5. ~A = {2, 4, 5}

### 6.5 笛卡尔积
A × B = {<1,a>, <1,b>, <2,a>, <2,b>, <3,a>, <3,b>}  
B × A = {<a,1>, <a,2>, <a,3>, <b,1>, <b,2>, <b,3>}

### 6.6 证明 A − (B ∪ C) = (A − B) ∩ (A − C)
∀x: x ∈ A−(B∪C) ⇔ x ∈ A ∧ x ∉ (B∪C) ⇔ x ∈ A ∧ x ∉ B ∧ x ∉ C  
⇔ (x ∈ A ∧ x ∉ B) ∧ (x ∈ A ∧ x ∉ C) ⇔ x ∈ (A−B) ∧ x ∈ (A−C) ⇔ x ∈ (A−B) ∩ (A−C)

### 6.7 容斥
|M| = 30, |P| = 25, |M∩P| = 10, |全班| = 50  
- 只学数学 = 30 − 10 = **20**
- 两门都学 = **10**
- 只学一门 = (30−10) + (25−10) = 20 + 15 = **35**
- 学至少一门 = 30 + 25 − 10 = 45
- 两门都不学 = 50 − 45 = **5**

### 6.8 分配律
∀x: x ∈ A∩(B∪C) ⇔ x ∈ A ∧ x ∈ (B∪C) ⇔ x ∈ A ∧ (x ∈ B ∨ x ∈ C)  
⇔ (x ∈ A ∧ x ∈ B) ∨ (x ∈ A ∧ x ∈ C) ⇔ x ∈ (A∩B) ∨ x ∈ (A∩C)  
⇔ x ∈ (A∩B) ∪ (A∩C)

### 6.9 三集合并
|A∪B∪C| = |A|+|B|+|C| − |A∩B|−|A∩C|−|B∩C| + |A∩B∩C|  
= 30+20+15 − 8−5−4 + 2 = 65 − 17 + 2 = **50**

### 6.10 国考 2017
若 A⊕B = A⊕C，则 (A∪B)−(A∩B) = (A∪C)−(A∩C)  
两边 ∪ A: (A∪B) = (A∪C)  
两边 ∩ A: A−B = A−C  
于是 A∪B = A∪C 且 A∩B = A∩C  
→ A−B = A−C  
→ B−A = C−A  
合并：B = C ✓

## 第7讲 二元关系 答案

### 7.1 恒等 / 全域
I_A = {<1,1>, <2,2>, <3,3>}  
E_A = {<1,1>, <1,2>, <1,3>, <2,1>, <2,2>, <2,3>, <3,1>, <3,2>, <3,3>}

### 7.2 合成
R = {<1,2>, <2,3>, <3,1>}, S = {<1,1>, <2,3>}  
- R∘S：找 <x,y>∈R, <y,z>∈S  
  <1,2>, <2,3> → <1,3> ∈ R∘S  
  <2,3>, <3,?> S 中无 <3,?>  
  <3,1>, <1,1> → <3,1> ∈ R∘S  
  R∘S = {<1,3>, <3,1>}
- S∘R：<1,1>, <1,2> → <1,2>  
  <2,3>, <3,1> → <2,1>  
  S∘R = {<1,2>, <2,1>}

### 7.3 性质判断
R = {<1,2>, <2,3>, <3,4>} on A = {1,2,3,4}
- **自反**？❌（缺 <1,1>, <2,2>, <3,3>, <4,4>）
- **反自反**？✓（所有 <x,x> 都不在 R 中）
- **对称**？❌（<1,2>∈R 但 <2,1>∉R）
- **反对称**？✓（没有 <x,y> 和 <y,x> 同时在 R 中）
- **传递**？✓（<1,2>,<2,3>→<1,3>∈R; <2,3>,<3,4>→<2,4>∉R ❌）  
  **❌ 不传递**

### 7.4 逆
R⁻¹ = {<2,1>, <1,2>, <3,2>, <2,3>, <3,3>}

### 7.5 关系图与性质
关系图：a→b, a→c, b→a, c→a, c→b, c→c
- 自反？❌（缺 <a,a>, <b,b>）
- 反自反？❌（<c,c>∈R）
- 对称？❌（<a,b>∈R 但 <b,a>∈R；但 <b,a>∈R 又 <a,b>∈R ✓ 这对；<a,c>∈R <c,a>∈R；<c,b>∈R 但 <b,c>∉R ❌）
- 反对称？✓（没有 <x,y>, <y,x> 同时在 R 中 x≠y）
- 传递？<b,a>∈R, <a,c>∈R → <b,c>∈R ✓；<b,a>∈R, <a,b>∈R → <b,b>∈R ❌  
  **不传递**

### 7.6 传递闭包
R = {<1,2>, <1,3>, <2,3>, <3,4>}
- R² = R∘R：<1,2><2,3> → <1,3>（已有）；<2,3><3,4> → <2,4>；<1,3><3,4> → <1,4>
- R² = {<1,3>, <2,4>, <1,4>} ∪ R = R ∪ {<2,4>, <1,4>}
- R³ = R²∘R：<1,4>无后继
- t(R) = R ∪ R² = {<1,2>, <1,3>, <2,3>, <3,4>, <2,4>, <1,4>}

### 7.7 对称 ⇔ R=R⁻¹
**⇒**：R 对称 ⇒ ∀<x,y>∈R, <y,x>∈R ⇒ <y,x>∈R⁻¹ ⇒ <x,y>∈R⁻¹ ⇒ R⊆R⁻¹  
反向类似 R⁻¹⊆R  
**⇐**：R=R⁻¹ ⇒ <x,y>∈R ⇒ <y,x>∈R⁻¹=R ⇒ 对称 ✓

### 7.8 整除关系
- **不是等价关系**（不反自反不反对称）
- **是偏序关系** ✓（自反、反对称、传递）
- [6]R = {1, 2, 3, 6}（约数）
- [8]R = {1, 2, 4, 8}

### 7.9 哈斯图 + 特殊元素
偏序集 <{1,2,3,4,6,12}, |>

### A10 综合大题
证明 P→Q ⇔ P→(P∧Q) ⇔ (P→Q)∧(P→¬Q) ⇔ P→(Q∧¬Q)

真值表：
| P | Q | P→Q | P→(P∧Q) | (P→Q)∧(P→¬Q) | P→(Q∧¬Q) |
|---|---|-----|---------|--------------|----------|
| 0 | 0 | T | T | T | T |
| 0 | 1 | T | T | T | T |
| 1 | 0 | F | F | F | F |
| 1 | 1 | T | T | F | F |

**注意**：第 4 列 P→(Q∧¬Q) 和第 3 列 (P→Q)∧(P→¬Q) 在 P=1,Q=1 时都是 F  
但 P→Q 和 P→(P∧Q) 在 P=1,Q=1 时是 T  

所以 **P→Q ⇔ P→(P∧Q)** 真  
但 (P→Q)∧(P→¬Q) ⇔ P→(Q∧¬Q) 真  
而 P→Q 和 (P→Q)∧(P→¬Q) **不等价**（P=1,Q=1 时不同）

等值变换：
- P→Q ⇔ ¬P∨Q
- P→(P∧Q) ⇔ ¬P∨(P∧Q) ⇔ (¬P∨P)∧(¬P∨Q) ⇔ 1∧(¬P∨Q) ⇔ ¬P∨Q ✓
- P→(Q∧¬Q) ⇔ ¬P∨0 ⇔ ¬P
- (P→Q)∧(P→¬Q) ⇔ (¬P∨Q)∧(¬P∨¬Q) ⇔ ¬P∨(Q∧¬Q) ⇔ ¬P ✓

**含义**：
- P→Q ⇔ P→(P∧Q)：P→Q 等价于"在 P 真的前提下还要 Q 才真"
- (P→Q)∧(P→¬Q) ⇔ P→(Q∧¬Q)：要求 P→Q 且 P→¬Q 等价于要求 Q∧¬Q（P 必假）
- 因此：(P→Q)∧(P→¬Q) ⇔ ¬P，不等价于 P→Q
