## 第3页 第1讲 命题逻辑基础

### 命题
- 定义：能唯一确定真假值的陈述句
- 例：① "2+2=3"（假）② "华盛顿是美国首都"（真）③ "我正在说假话"（悖论，非命题）
- 非命题：疑问句、祈使句、感叹句、含自由变量的句子、悖论

### 5 个常用联结词
| 符号 | 名称 | 含义 |
|---|---|---|
| ¬ | 否定 | 非 P |
| ∧ | 合取 | P 且 Q |
| ∨ | 析取 | P 或 Q（相容或） |
| ⊕ 或 ⊻ | 异或 | P、Q 不可同真 |
| → | 蕴涵 | 如果 P 则 Q（P 为 Q 的充分条件，Q 为 P 的必要条件） |
| ↔ | 等价 | P 当且仅当 Q |

### 真值表要点
- P→Q：仅当 P=T、Q=F 时为 F，其余 T
- P↔Q：P、Q 同真值时为 T
- 优先级：¬ > ∧ > ∨ > → > ↔

### 合式公式（wff）
1. 单个命题变元是 wff
2. 若 A 是 wff，则 ¬A 是 wff
3. 若 A、B 是 wff，则 (A∧B)/(A∨B)/(A→B)/(A↔B) 是 wff
4. 有限次使用 ①②③ 得到的是 wff

### 公式分类
- **永真式（重言式）**：所有赋值下恒 T（例：¬P∨P、P∨¬P）
- **永假式（矛盾式）**：所有赋值下恒 F（例：¬P∧P）
- **可满足式**：至少有一种赋值为 T
- **一般式**：既非永真也非永假

## 第4页 第2讲 等值式（核心）

### 等价定义
若 A、B 在所有真值指派下真值相同，则 A⇔B。等值定理：A⇔B 当且仅当 A↔B 是永真式。

### 24 个基本等值式
| 律名 | 公式 |
|---|---|
| 对合律 | ¬¬P = P |
| 幂等律 | P∨P=P；P∧P=P |
| 结合律 | (P∨Q)∨R=P∨(Q∨R)；(P∧Q)∧R=P∧(Q∧R) |
| 交换律 | P∨Q=Q∨P；P∧Q=Q∧P |
| 分配律 | P∨(Q∧R)=(P∨Q)∧(P∨R)；P∧(Q∨R)=(P∧Q)∨(P∧R) |
| 吸收律 | P∨(P∧Q)=P；P∧(P∨Q)=P |
| 德摩根律 | ¬(P∨Q)=¬P∧¬Q；¬(P∧Q)=¬P∨¬Q |
| 同一律 | P∨F=P；P∧T=P |
| 零律 | P∨T=T；P∧F=F |
| 否定律 | P∨¬P=T；P∧¬P=F |

### 化归公式（必背）
- P→Q ⇔ ¬P∨Q
- P→Q ⇔ ¬Q→¬P（逆否等价）
- P↔Q ⇔ (P→Q)∧(Q→P)
- P↔Q ⇔ (P∧Q)∨(¬P∧¬Q)
- P↔Q ⇔ ¬Q↔¬P

### 证明方法
1. 真值表（变量 ≤3 时优先）
2. 等值变换 + 置换定律
3. 代入不变性 + 等值关系的传递性

### 等价式性质
自反性、对称性、传递性、置换保持等价

## 第5页 第2讲 重言蕴涵与推理

### 重言蕴涵式 A⇒B
若 A→B 是永真式，则 A 永真蕴涵 B。**证明三法**：
1. 真值表
2. 假设前件为真 → 推出后件为真
3. 假设后件为假 → 推出前件为假

### 16 个重要重言蕴涵式
| 公式 | 推理模式 |
|---|---|
| P∧Q ⇒ P；P∧Q ⇒ Q | 化简律 |
| P ⇒ P∨Q；Q ⇒ P∨Q | 附加律 |
| ¬P ⇒ P→Q；Q ⇒ P→Q | 假前件/真后件律 |
| ¬(P→Q) ⇒ P；¬(P→Q) ⇒ ¬Q | 蕴涵否定律 |
| P,Q ⇒ P∧Q | 合取引入律 |
| ¬P∧(P∨Q) ⇒ Q | **析取三段论** |
| P∧(P→Q) ⇒ Q | **假言推理 MP** |
| ¬Q∧(P→Q) ⇒ ¬P | **拒取式 MT** |
| (P→Q)∧(Q→R) ⇒ P→R | **假言三段论 HS** |
| (P∨Q)∧(P→R)∧(Q→R) ⇒ R | **构造性二难** |
| A→B ⇒ (A∨C)→(B∨C)；A→B ⇒ (A∧C)→(B∧C) | 单调性 |

### 推理规则
- **P**（引入前提）：可随时引用前提
- **T**（引入结论）：若前步有 X⇒Y，可引入 Y
- **CP**（条件论证）：若 H∧R ⇒ S，则 H ⇒ R→S

### 直接推理格式
```
(1) P        P    前提
(2) P→Q      P    前提
(3) Q        T (1)(2) I    假言推理
```

## 第6页 第3讲 范式

### 范式（Normal Form）
命题公式的规范形式，约定只用联结词 ¬、∨、∧。

### 基本概念
- **简单合取式**：用 ∧ 联结命题变元或其否定
- **简单析取式**：用 ∨ 联结命题变元或其否定
- **析取范式**：A₁∨A₂∨…∨Aₙ，每个 Aᵢ 为合取式
- **合取范式**：A₁∧A₂∧…∧Aₙ，每个 Aᵢ 为析取式

### 求法步骤
1. 化归去掉 →、↔
2. 德摩根律把 ¬ 后移到命题变元前
3. 分配律、结合律、幂等律整理

### 主析取范式（DNF）
- **小项 mᵢⱼ**：n 个变元的合取式，每个变元或否定恰出现一次（共 2ⁿ 个）
- **性质**：每组赋值有且仅有 1 个小项为 T
- **求法 Ⅰ**：真值表中公式值为 T 的赋值所对应小项的析取
- **求法 Ⅱ**：先用等价变换得析取范式，再用 (R∨¬R)=T 补齐变元
- 例：P→Q = m₀₀∨m₀₁∨m₁₁ = (¬P∧¬Q)∨(¬P∧Q)∨(P∧Q)

### 主合取范式（CNF）
- **大项 Mᵢⱼ**：n 个变元的析取式，每个变元或否定恰出现一次
- **性质**：每组赋值有且仅有 1 个大项为 F
- **求法**：真值表中公式值为 F 的赋值所对应大项的合取
- 例：P→Q = M₁₀ = ¬P∨Q

### 极小联结词完备集
{¬, ∧}、{¬, ∨}、{¬, →}、{↑(与非)}、{↓(或非)} 均是完备集

## 第7页 第4讲 谓词逻辑（一阶逻辑）

### 为什么需要谓词逻辑
命题逻辑的局限：无法表达含个体与总体的推理  
例："人固有一死，苏格拉底是人，因此苏格拉底固有一死"  
- P(x)：x 是人；D(x)：x 要死；S：苏格拉底
- (∀x(P(x)→D(x))) ∧ P(S) ⇒ D(S)

### 基本概念
- **个体词**：研究对象。a,b,c 为个体常项；x,y,z 为个体变项
- **个体域**：个体变项的取值范围（特指/全总）
- **谓词**：刻画个体性质或关系。大写字母表示，n 元谓词
- **量词**：
  - ∀：全称量词（"所有"）
  - ∃：存在量词（"存在"）

### 谓词公式的形成规则
1. n 元谓词是公式
2. ¬A 是公式
3. (A∨B)/(A∧B)/(A→B)/(A↔B) 是公式
4. ∀xA(x)、∃xA(x) 是公式
5. 有限次使用以上规则得到的也是公式

### 量词等值定律
- ¬∀xA(x) ⇔ ∃x¬A(x)
- ¬∃xA(x) ⇔ ∀x¬A(x)
- ∀x(A(x)∧B(x)) ⇔ ∀xA(x)∧∀xB(x)
- ∃x(A(x)∨B(x)) ⇔ ∃xA(x)∨∃xB(x)
- ∀x(P∨B(x)) ⇔ P∨∀xB(x)（P 不含 x）
- ∃x(P∧B(x)) ⇔ P∧∃xB(x)

### 量词顺序
**不能随便交换**！∀x∃yL(x,y) 与 ∃y∀xL(x,y) 含义不同。

### 推理规则
- UI（全称特化）：∀xA(x) ⇒ A(c)
- EG（存在推广）：A(c) ⇒ ∃xA(x)
- 量词分配律 + MP/MT/HS 组合使用

## 第10页 第5讲 集合及其运算

### 集合的定义与表示
- 集合：不同对象汇集的整体
- 表示法：列举法、描述法、Venn 图
- 关系：∈（属于）、⊆（包含）、⊂（真包含）、=（相等）

### 基本运算
| 运算 | 符号 | 定义 |
|---|---|---|
| 并 | A∪B | {x \| x∈A ∨ x∈B} |
| 交 | A∩B | {x \| x∈A ∧ x∈B} |
| 差 | A−B | {x \| x∈A ∧ x∉B} |
| 对称差 | A⊕B | (A−B)∪(B−A) = (A∪B)−(A∩B) |
| 绝对补 | ~A | {x \| x∉A}（全集下） |
| 笛卡尔积 | A×B | {<x,y> \| x∈A ∧ y∈B} |

### 集合恒等式（核心）
- 交换律：并、交
- 结合律：并、交
- 分配律：并对交、交对并
- 德摩根律：~(A∪B)=~A∩~B；~(A∩B)=~A∪~B
- 吸收律：A∪(A∩B)=A；A∩(A∪B)=A
- 同一律：A∪∅=A；A∩U=A
- 补律：A∪~A=U；A∩~A=∅

### 容斥原理
|A₁∪A₂∪…∪Aₙ| = Σ|Aᵢ| − Σ|Aᵢ∩Aⱼ| + Σ|Aᵢ∩Aⱼ∩Aₖ| − … + (−1)ⁿ⁺¹|A₁∩…∩Aₙ|

### 集合的基数
- 有穷集：|A| = 元素个数
- 子集数：|A|=n 时 P(A) 有 2ⁿ 个子集

## 第11页 第6讲 二元关系（上）

### 笛卡尔积
A×B = {<x,y> \| x∈A ∧ y∈B}  
- A×B ≠ B×A（一般）
- 若 A=∅ 或 B=∅，则 A×B=∅

### 二元关系定义
- A×B 的任意子集 R 称为 A 到 B 的二元关系
- A=A 时，R 称为 A 上的二元关系
- |A|=n 时，A 上可定义 2^(n²) 个二元关系

### 三种表示
- **集合表示**：R={<x,y> \| …}
- **关系图**：顶点表元素，有向边表有序对
- **关系矩阵**：M_R = (rᵢⱼ)，rᵢⱼ=1 表示 <aᵢ,aⱼ>∈R

### 基本关系
- 全域关系 E_A：R = A×A
- 恒等关系 I_A：R = {<x,x> \| x∈A}（单位矩阵）
- 空关系 ∅

### 关系的运算
- 逆：R⁻¹ = {<y,x> \| <x,y>∈R}
- 合成：R∘S = {<x,z> \| ∃y(<x,y>∈R ∧ <y,z>∈S)}
- 限制：R↾A = {<x,y> \| <x,y>∈R ∧ x∈A}
- 像：R[A] = {y \| ∃x∈A, <x,y>∈R}
- 幂：R⁰=I_A，R^(n+1)=Rⁿ∘R

### 性质定理
| 性质 | 等价条件 |
|---|---|
| 自反 | I_A ⊆ R |
| 反自反 | R ∩ I_A = ∅ |
| 对称 | R = R⁻¹ |
| 反对称 | R ∩ R⁻¹ ⊆ I_A |
| 传递 | R∘R ⊆ R |

## 第12页 第6讲 二元关系（下）

### 闭包
- **自反闭包**：r(R) = R ∪ I_A
- **对称闭包**：s(R) = R ∪ R⁻¹
- **传递闭包**：t(R) = R ∪ R² ∪ R³ ∪ … ∪ Rᵏ（k≤|A|）

### 等价关系
满足自反、对称、传递的关系。  
- 等价类：[x]_R = {y \| xRy}
- 商集：A/R = {[x]_R \| x∈A}
- 划分 π 与等价关系 R 一一对应

### 偏序关系
满足自反、反对称、传递的关系，记作 ≼。
- 偏序集 <A,≼>
- 全序（线序）：任意两元素可比
- 覆盖：x≺y 且不存在 z 使 x≺z≺y
- **哈斯图**：省去自反环，用位置高低表示序，仅覆盖关系连边

### 特殊元素
| 元素 | 定义 |
|---|---|
| 极大元 | 无大于它的元素 |
| 极小元 | 无小于它的元素 |
| 最大元 | ≥ 所有元素 |
| 最小元 | ≤ 所有元素 |
| 上界 | ≥ 子集所有元素 |
| 下界 | ≤ 子集所有元素 |
| 上确界 | 最小上界（若存在则唯一） |
| 下确界 | 最大下界（若存在则唯一） |

注：最小元⇔下确界；最大元⇔上确界。但反之不成立。

## 第13页 函数（集合论视角）

### 定义
f: A→B 是满足单值性的二元关系：∀x∈A, ∃!y∈B, <x,y>∈f

### 性质
- **单射**（入射）：x₁≠x₂ ⇒ f(x₁)≠f(x₂)
- **满射**：f(A)=B（值域 = 陪域）
- **双射**：既是单射又是满射 ⇔ 可逆

### 运算
- 复合：(f∘g)(x) = f(g(x))，右结合
- 逆：仅双射函数有逆 f⁻¹

### 等势（用于第7讲铺垫）
若 ∃ 双射 f: A→B，则 A≈B。  
- ≈ 满足自反、对称、传递
- Z≈N、Q≈N、N×N≈N
- 任何实数区间都与 R 等势
- A≉P(A)（康托定理）

## 第14页 第7讲 集合的基数

### 等势与优势
- 等势 A≈B：∃ 双射 f: A→B
- 优势 A≼·B：∃ 单射 f: A→B
- 真优势 A≺·B：A≼·B 且 A≠B

### 重要结果
**等势**：N ≈ Z ≈ Q ≈ N×N ≈ P(N)（直觉上"可数无穷"）  
**等势**：R ≈ [a,b] ≈ (c,d) ≈ {0,1}^N ≈ P(N)  
**真优势**：N≺R；A≺P(A)（康托定理）

### 有穷与无穷
- A 有穷 ⇔ A 等势于某自然数 n（n=|A|）
- N、R 都是无穷集

### 基数
- 有穷集 A：card A = |A|（与 A 等势的惟一天然数）
- 自然数集 N：card N = ℵ₀（阿列夫零）
- 实数集 R：card R = ℵ（阿列夫）
- ℵ₀ < ℵ

### 可数集
- 定义：card A ≤ ℵ₀
- 性质：
  - 可数集的子集仍可数
  - 两个可数集的并可数
  - 可数集 × 可数集 可数
  - 可数个可数集的笛卡儿积可数
  - 无穷集的幂集不可数

### 连续统问题
N ≤ A ≤ R 但 A 与 R 不等势？  
- 1874 提出，1963 科恩证明与 ZFC 公理体系独立
- 连续统假设：不存在严格介于 ℵ₀ 和 ℵ 之间的基数

## 第16页 第8讲 代数结构之群论引言

### 伽罗华与群论
- 1811-1832，巴黎人，21 岁死于决斗
- 群论的开创者，解决了高次方程可根式求解的判定
- 精神：置换的不变量是群论的核心思想

### 群（Group）的定义
非空集合 G 上定义"乘法"运算 ×，满足：
1. **封闭性**：∀a,b∈G, a×b∈G
2. **结合律**：∀a,b,c∈G, (a×b)×c = a×(b×c)
3. **单位元**：∃e∈G, ∀a∈G, e×a = a×e = a
4. **逆元**：∀a∈G, ∃a⁻¹∈G, a×a⁻¹ = a⁻¹×a = e

### 群的分类
- **Abel 群**：满足交换律；否则非 Abel
- **有限群 / 无限群**：按 |G|
- 例：
  - ({1,−1}, ×) 是 2 阶 Abel 群
  - ({0,1,…,n−1}, +ₙ) 记作 Zₙ
  - (Z, +), (Q, +), (R, +) 是无限 Abel 群
  - GL(n,R) = n 阶可逆矩阵全体关于乘法构成非 Abel 群

### 基本性质
1. 单位元唯一
2. 每个元素的逆元唯一
3. 消去律：ab=ac ⇒ b=c；ba=ca ⇒ b=c
4. 元素 a 的阶 o(a)：使 aʳ=e 的最小正整数 r

## 第17页 第8讲 置换与置换群

### n 次置换
集合 X={1,2,…,n} 到自身的一一对应 f: i ↦ aᵢ，记作：
$$f = \begin{pmatrix} 1 & 2 & \cdots & n \\ a_1 & a_2 & \cdots & a_n \end{pmatrix}$$

### 置换的运算（复合）
(f₁f₂)(i) = f₁(f₂(i)) — 先施行 f₂，再 f₁  
X 上所有 n 次置换构成 n 次对称群 Sₙ，|Sₙ|=n!

### 轮换表示
任何置换可唯一分解为不相交轮换之积：
- 例：f = (1 3 2)(4 5)(6) 即不相交轮换
- (123) = (231) = (312)（圆排列等价）

### 奇偶性
- 对换 (i,j) 是 2 阶轮换
- 任何置换 = 若干对换之积
- **奇偶性不变**：若分解为对换的个数奇偶性是置换的本质属性
- **奇置换 / 偶置换**：含奇数/偶数个对换

### 交错群 Aₙ
Sₙ 中所有偶置换构成 n 次交错群 Aₙ，|Aₙ|=n!/2（n≥2）  
且 Aₙ 是 Sₙ 的子群。

### 应用：14-15 滑块谜题
判断能否通过 0 与相邻数字的换位从布局 A 达到布局 B  
- 看 A→B 对应的置换的奇偶性
- 0 走回原位需偶数次换位
- 奇置换 ⇒ 不可能

## 第19页 第9讲 排列组合（补充讲次）

### 研究对象
组合数学关心"按一定方式配置一组事物"，主要考虑：
| 方面 | 问题 | 典型例 |
|---|---|---|
| 存在性 | 配置是否存在？ | 8×8 棋盘挖去 1 黑 1 白能否骨牌覆盖 |
| 计数与分类 | 有多少种？ | 八面体顶点染色方案数 |
| 构造算法 | 如何构造？ | Hamilton 圈近似算法 |
| 算法优化 | 哪种最优？ | 时空复杂度对比 |

### 两个基本计数原理
- **加法原理**（分类）：m₁+m₂+…+mₙ（事件互相独立）
- **乘法原理**（分步）：m₁×m₂×…×mₙ（步骤互相独立）

### 排列与组合
| 名称 | 公式 | 关键 |
|---|---|---|
| **排列** P(n,r) | n!/(n-r)! | 按次序 |
| **组合** C(n,r) | n!/(r!(n-r)!) | 不计次序 |
| **圆周排列** Q(n,r) | P(n,r)/r | 旋转等价 |

**关键关系**：P(n,r) = r! × C(n,r)

### 经典例题
- 例 1.4：6 字母组成 4 字母单词
  - 不重复：P(6,4) = 360
  - 可重复：6⁴ = 1296
- 例 1.5：{1,…,9} 选数字，4,5,6 不相邻的 7 位数
  - 7 位总数 P(9,7) = 181440
  - 4,5,6 相邻数 = 6×5×P(6,4) = 10800
  - 不相邻 = **181440 − 10800 = 17064**
- 例 1.7：5 对夫妻围圆桌夫妻相邻 = **4! × 2⁵ = 1024**
- 广场 6 入口 9 辆车入场方式 = **14!/5!**

### 牛顿二项式公式
$$(x+y)^n = \sum_{k=0}^{n} \binom{n}{k} x^k y^{n-k}$$

推广：
- $(1+x)^{-n} = \sum_{k=0}^{\infty} (-1)^k \binom{n+k-1}{k} x^k$
- $(1-x)^{-n} = \sum_{k=0}^{\infty} \binom{n+k-1}{k} x^k$

### 六个组合数学典型问题
1. **分派问题**：15 个方块分到 25 格（10 空）= 25!/(25-15)! = 25×24×…×11
2. **染色问题**：正方形 4 顶点 3 色 = 3⁴ = 81 = (r+b+g)⁴ 展开式各项系数和
3. **36 军官问题**：6 阶正交拉丁方不存在（1901 Tarry）；n=4k+2 (k≥2) 可构造
4. **中国邮路问题**（管梅谷 1962）：偶度点 Euler 闭迹；奇度点配对加重复边
5. **四色问题**（1850 提出，1976 证明）：地图相邻国家异色只需 4 色
6. **棋盘覆盖**：m×n 棋盘完全覆盖 ⇔ m、n 均偶；标准 8×8 共 **12,988,816** 种覆盖

### 一一对应
计数 A 难而 B 易时，若 A、B 一一对应，则 |A| = |B|。

## 第20页 综合练习与典型题

### 题型 1：命题逻辑推理
```
例：P→Q, Q→R, P ⇒ R（假言三段论）
(1) P        P
(2) P→Q      P
(3) Q        T(1)(2) I
(4) Q→R      P
(5) R        T(3)(4) I
```

### 题型 2：主析取范式与主合取范式
```
例：P→Q 的主析取范式 = m₀₀∨m₀₁∨m₁₁
                   = (¬P∧¬Q)∨(¬P∧Q)∨(P∧Q)
    P→Q 的主合取范式 = M₁₀ = ¬P∨Q
```

### 题型 3：谓词逻辑推理
```
例：苏格拉底三段论
前提：∀x(P(x)→D(x))，P(S)
求证：D(S)
证明：
(1) ∀x(P(x)→D(x))   P
(2) P(S)             P
(3) P(S)→D(S)       T(1) UI
(4) D(S)             T(2)(3) I
```

### 题型 4：关系性质判断
- 看对角线：自反 / 反自反
- 看对称位置：对称 / 反对称
- 看传递性：M²⊆M（传递闭包判断）

### 题型 5：哈斯图与特殊元素
- 自反环去掉
- 仅画覆盖关系的边
- 用高度表示序
- 极大/极小/最大/最小/上确界/下确界辨识

### 题型 6：集合基数
- card N = ℵ₀，card R = ℵ
- 判定可数性：找双射或用可数集性质
- 连续统问题（了解即可）

### 题型 7：置换计算与奇偶性
- 分解为轮换
- 判断奇偶性
- 应用：n 阶魔方、15-谜题

## 第21页 考前速记卡

### 命题逻辑
- **6 个联结词**：¬ ∧ ∨ ⊕ → ↔
- **24 个等值式**：对幂结交分吸摩同零否
- **化归**：→⇔¬∨；↔⇔双→；逆否等价
- **5 大推理模式**：化简、附加、合取引入、MP、MT、HS、构造性二难
- **CP 规则**：附加前提 → 推出结论 → 包成蕴含

### 谓词逻辑
- **量词转换**：¬∀=∃¬；¬∃=∀¬
- **量词分配**：∀ 对 ∧；∃ 对 ∨（方向不可反）
- **量词顺序**：∀∃ ≠ ∃∀

### 集合与关系
- **集合运算**：∪ ∩ − ⊕
- **容斥**：加减交替
- **关系表示**：集合/图/矩阵 三合一
- **5 性质**：自反/反自反/对称/反对称/传递
- **闭包公式**：r(R)=R∪I；s(R)=R∪R⁻¹；t(R)=R∪R²∪…∪Rᵏ
- **偏序 3 性质**：自反+反对称+传递
- **哈斯图**：去环、位置高低、仅覆盖连边

### 集合的基数
- **N≈Z≈Q≈N×N**：可数无穷，基数 ℵ₀
- **R≈P(N)≈[a,b]**：连续统，基数 ℵ
- **ℵ₀ < ℵ**
- **可数集性质**：子集、并、笛卡儿积仍可数

### 代数结构
- **群 4 条件**：封闭、结合、单位、逆
- **|Sₙ|=n!；|Aₙ|=n!/2**
- **奇偶性**：分解为对换个数奇偶不变

### 解题步骤
1. **审题**：先识别是哪个模块
2. **符号化**：自然语言 → 命题/谓词/集合
3. **方法选择**：真值表 / 等值演算 / 推理规则 / 矩阵
4. **检查**：用性质公式反向验证
